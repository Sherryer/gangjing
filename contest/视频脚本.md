# 杠精虾 · 3分钟演示视频脚本

> 总时长：约 3 分钟
> 形式：录屏 + 剪辑 + 字幕旁白 + 背景音乐
> 风格：暗色终端 / 节奏感 / 有点狠

---

## 剪辑整体节奏

```
0:00 - 0:40   第一幕：悬念（纯旁白，不出现界面）
0:40 - 2:20   第二幕：爆发（录屏 + 剪辑，核心 Demo）
2:20 - 3:00   第三幕：收尾（HTML 报告展示 + slogan 落版）
```

---

## 第一幕：悬念（0:00 - 0:40）

### 画面
黑屏。白色细体字逐句淡入，节奏慢，像打字机。

### 字幕文案（逐句出现，每句停留 2-3 秒）

```
你写完的代码，自己看了三遍，觉得没问题。

你写完的方案，自我感觉逻辑严密。

然后上线。

翻车。
```

*停顿 1 秒*

```
原因往往只有一个——

你周围没有一个真的会说"不行"的人。
```

*停顿 1 秒，背景音乐渐入（低频鼓点）*

```
直到现在。
```

*画面切入：终端黑屏，光标闪烁*

---

## 第二幕：爆发（0:40 - 2:20）

> ⚠️ 录屏建议：字体 22px+，暗色主题（One Dark / Dracula），终端全屏

### 第一段：代码 Review（0:40 - 1:20）

**[录屏片段 1]** 输入命令，回车

```bash
python main.py --file demo_code.py --type code --level 2
```

**[字幕]** `丢进去一段"看起来没问题"的登录模块`

---

**[录屏片段 2]** Banner 出现，Spinner 转动

*画面：🦐 杠精虾 Banner 闪出，红色边框，下方 dots spinner 转动，计时器跳动*

**[字幕]** `它没有情绪，只有一个目标`

---

**[录屏片段 3 - 关键帧，放慢 0.7x]** Review 结果出现

*画面：Rich 终端，红色 "Review 结果" 分割线，P0 × 2 红色醒目*

**[字幕]** `P0 × 2  —  SQL 注入漏洞 / 弱密钥硬编码`

---

**[录屏片段 4 - 加速到 P0 列表]** 具体问题列表滚动

*画面：P0 红色 Badge、P1 橙色 Badge，每条精准定位到行号和改法*

**[字幕]** `不只告诉你有问题，还告诉你第几行，怎么改`

---

### 第二段：三虾互杠（1:20 - 2:00）

**[录屏片段 5]** 切换命令

```bash
python main.py --mode three --file demo_business.md --type business --level 3
```

**[字幕]** `换一份商业计划书，开三虾互杠模式`

---

**[录屏片段 6 - 加速 2x]** 迭代表格出现

*画面：三虾迭代记录表格*
```
Round 1   P0×3   P1×4   🔄 继续
Round 2   P0×1   P1×2   🔄 继续
Round 3   P0×0   P1×1   ✅ 通过
```

**[字幕]** `三轮迭代，P0 从 3 降到 0，自动收敛`

---

### 第三段：HTML 导出（2:00 - 2:20）

**[录屏片段 7]** 加上 --html --open 参数，回车

```bash
python main.py --file demo_business.md --type business --level 3 --html --open
```

**[字幕]** `一个参数，报告直接在浏览器打开`

---

**[录屏片段 8 - 切换到浏览器，慢推]** HTML 报告全貌

*画面：红色渐变 Header，P0/P1 彩色 Badge，三虾历史折叠展开*

**[字幕]** `自包含 HTML，离线可用，直接交给甲方`

---

```

```

```
🦐  杠精虾
```

```
你的同事只会点赞，虾不会。
```

*停留 3 秒，淡出*

## 录屏操作清单（按顺序录）

### 录制前准备
- [ ] 终端字体：22px，等宽字体（SF Mono / Fira Code / Cascadia Code）
- [ ] 终端主题：One Dark Pro 或 Dracula（暗色背景，红色系更好看）
- [ ] 终端窗口：全屏，隐藏标题栏
- [ ] 确保 API Key 有效，网络通畅
- [ ] 预跑一次确认输出好看（有 P0 有 P1，三虾模式能跑到 Round 2-3）

### 录制片段 1：代码 Review
```bash
# 在项目根目录
python main.py --file contest/demo/demo_code.py --type code --level 2 --html --open
```
- 录完整流程：命令输入 → Banner → Spinner → 结果输出 → HTML 弹出
- 原始素材约 1-2 分钟

### 录制片段 2：三虾模式
```bash
python main.py --mode three --file contest/demo/demo_business.md --type business --level 3 --html --open
```
- 录完整流程：命令输入 → Banner → 迭代进行中 → 汇总表格 → 结果输出 → HTML 弹出
- 原始素材约 3-5 分钟（三虾比较慢）

## 剪辑备注

| 片段 | 剪辑处理 |
|------|---------|
| Spinner 转动 | 保留 3-5 秒即可，剪掉等待时间 |
| LLM 输出滚动 | 加速到 1.5-2x，保留关键 P0/P1 段落放慢 |
| 三虾迭代过程 | 可以直接跳切到迭代汇总表格 |
| HTML 报告展示 | 慢速滚动，配合旁白节奏，重点展示 P0 红色区域 |

**背景音乐建议：**
- Lo-fi / 低鼓点电子乐，有紧张感但不抢镜
- 推荐：YouTube Audio Library 搜索 "cinematic tension" 或 "dark electronic"
- 第一幕无音乐 → 0:30 渐入 → 第二幕全程 → 第三幕淡出
