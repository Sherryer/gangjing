# 内容/文案审查 Skill
# 适用类型：article / copywriting / announcement / report
# 版本：v1.0

## 额外审查维度（在通用检查清单之上）

| 维度 | 图标 | 关注点 |
|-----|------|-------|
| 事实准确性 | 🔴 | 引用数据是否正确、事实陈述是否可验证、信源可靠性 |
| 逻辑严密性 | 🟡 | 论证链条是否完整、有无偷换概念/稻草人谬误/false dilemma |
| 立场偏差 | 🟣 | 是否有未声明的利益关系、是否有春秋笔法/选择性引用 |
| 受众适配 | 🔵 | 语言风格是否匹配目标受众、专业术语是否需要解释 |
| 表达质量 | 🟢 | 结构是否清晰、是否有冗余/重复、标题是否准确反映内容 |

## P 级标准（内容/文案专用）

- P0 🔴：事实性错误 / 严重误导 / 可能引发法律问题
- P1 🟠：逻辑漏洞 / 重要遗漏 / 明显立场偏差未声明
- P2 🟡：表达可以更精准 / 论证可以更充分
- P3 🔵：措辞 / 格式 / 结构的微调建议

## 内容审查专用规则

1. 对每个数据引用追问"来源是什么？能否独立核实？"
2. 检查标题和正文是否一致，有无标题党/夸大
3. 检查是否有未声明的前提假设影响结论
4. 识别情绪化语言：是否在用情绪替代论证
5. 检查段落逻辑顺序：前后是否有因果/递进关系
